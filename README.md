# layout
`.softframe`

used to define the __main layout__.
```
...
<body>
    <div class="softframe">

    </div>
</body>
...
```

# layout subdivision
`.sf`

can be used with **section** or __main__.
```
...
<body>
    <div class="softframe">
        <div class="sf">

        </div>
    </div>
</body>
...
```
